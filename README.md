* This is an overhaul of my original BoomboxOverhaul mod. The original mod was designed to work with a specific YouTube boombox mod that is no longer maintained. Because of this I decided to implement a working, albeit sometimes temperamental YouTube playback.

# BoomBoxOverhaulV2 Features!

* Play YouTube links as audio via an in game UI.
* Server wide or Local only Audio Changes.
* Options for the boombox to have an infinite battery!
* Options for the boombox to be as light as a feather (Weightless).
* Selectable Preset Audio modes, information available [Here]()

## Configuration

* Before we begin I will let you know that there are many configurable options with this mod! The configuration file is generated as ```henreh.boomboxoverhaul.cfg```.
* Inside this file are three key Keybind settings - UI open Key, Volume Up Key and Volume down Key.
* Default values are: UI Open Key **[B]** Volume Up Key **[=]** Volume Down Key **[-]**.
* Also inside this configuration are 8 Categories:
Audio, Cache, Dependencies, Downloads, Gameplay, Input, Networking and Playlist. 
* Feel free to have browse to your hearts desire!

## Using this mod

* Hold a boombox and press `B` **```(Default bind)```** to open the URL UI.
* Paste a YouTube video or playlist URL. **```(Max time of a singular video by default configuration value is 1800 seconds - 30 minutes).```**
* Press `Play` and let the magic commence.
* The UI and tooltip (Hud element thingy when holding the boombox) will display a status message. Let the track prepare and play.
* The UI will display an error if ```YT-DL``` is unable to process the link.

## Notes

- This mod downloads and uses `yt-dlp` and `ffmpeg` when required.
- The boombox HUD shows the current track title.
- All players should have the mod installed for it to work.
